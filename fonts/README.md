# Fonts

Bitte folgende WOFF2-Dateien hier ablegen:

- `inter-variable.woff2` — Inter (normal), Variable Font
- `inter-variable-italic.woff2` — Inter (italic), Variable Font

Download: https://fonts.google.com/specimen/Inter  
Alternativ: https://google-webfonts-helper.herokuapp.com/fonts/inter

Bis die Dateien abgelegt sind, fällt der Browser auf System-Fonts zurück.
